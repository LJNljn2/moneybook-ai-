## Codebase Patterns
- UniApp (Vue3 + Vite + TypeScript) project scaffolded from `dcloudio/uni-preset-vue#vite-ts`
- Use `npm run type-check` for TypeScript validation (`vue-tsc --noEmit`)
- Use `npm run dev:h5` for H5 dev server, `npm run build:h5` for production build
- Project uses `src/` as source root with `pages/`, `components/`, `utils/`, `store/` directories
- Page routing configured in `src/pages.json` (not vue-router)
- Chat UI uses scroll-view with `scrollTop` bound to a ref; set to large value to scroll to bottom
- AI streaming: `chatCompletionStream` for H5, `chatCompletion` fallback for other platforms
- On non-tabBar pages, use `uni.navigateTo`; on tabBar pages, use `uni.switchTab`
- For lightweight charts in UniApp H5, use custom SVG components instead of external chart libraries — reactive via Vue computed properties, no npm dependencies
- For APP/Android builds: `npm run dev:app` (dev) or `npx uni build -p app` (prod); APK packaging requires HBuilderX cloud packaging or Android SDK
- App config lives in `src/manifest.json` — set `name`, `appid`, `distribute.android` for Android builds
